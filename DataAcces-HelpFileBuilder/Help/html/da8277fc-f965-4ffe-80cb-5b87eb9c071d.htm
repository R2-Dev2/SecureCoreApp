Technical Documentation - Data Acces

This is the documentation file where you can find all the information about the DataAcces library.

Getting Started

To start, you must add the DataAcces library to your project and configure the connection to your database. Once that is done you can start calling the methods to receive the data. The following default items are included in this project:

  • AccesADades.AccesADadest - Sets the necessary configuration to be able to connect to the database.

  • AccesADades.Actualitzar - Updates database values

  • AccesADades.Executa - Executes SQL Query in Databases.

See the DataAcces in the menu for documentation on how to use all the functions contained in the DataAcces library
